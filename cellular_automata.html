Cellular Automata

I learned about cellular automata and got fascinated by it. These simple cells on a grid with simple binary rules can give rise to extreme complexity and order. These simple entities can emulate Turing Machines, RNA and DNA structure. It also has relationships with fractal geometry and I am fascinated by it and these ideas as I learn more about it's power. I also wrote some code in Python for elementary CA but it has to go through a lot of refinements as I go deeper into this.
